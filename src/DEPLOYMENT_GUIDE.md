# 🚀 Руководство по развертыванию виртуального тура

## Для разработчиков (текущий способ)

Сейчас проект запускается через командную строку с помощью Node.js и Vite:

```bash
npm install
npm run dev
```

## Для обычных пользователей (простое решение)

Есть несколько способов упростить доступ к проекту для конечных пользователей:

---

## ✅ **Вариант 1: Развертывание на бесплатном хостинге (РЕКОМЕНДУЕТСЯ)**

Это самый простой способ — разверните проект на бесплатном хостинге и делитесь ссылкой.

### **A. Netlify (самый простой)**

1. Зарегистрируйтесь на [netlify.com](https://netlify.com)
2. Перетащите папку проекта в окно Netlify (Drag & Drop)
3. Netlify автоматически соберёт проект
4. Получите ссылку вида: `https://your-tour.netlify.app`
5. Поделитесь этой ссылкой с пользователями

**Плюсы:**
- ✅ Совершенно бесплатно
- ✅ Автоматическая сборка проекта
- ✅ HTTPS из коробки
- ✅ Можно использовать своё доменное имя

### **B. Vercel**

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. Подключите ваш GitHub репозиторий или загрузите проект
3. Vercel автоматически определит Vite и соберёт проект
4. Получите ссылку вида: `https://your-tour.vercel.app`

**Плюсы:**
- ✅ Бесплатно
- ✅ Отличная производительность
- ✅ Автоматические обновления при изменениях

### **C. GitHub Pages**

1. Создайте репозиторий на GitHub
2. Загрузите проект в репозиторий
3. В настройках репозитория включите GitHub Pages
4. Настройте GitHub Actions для автоматической сборки
5. Получите ссылку: `https://username.github.io/project-name`

**Плюсы:**
- ✅ Бесплатно
- ✅ Интеграция с Git
- ✅ Автоматическое развертывание

---

## ✅ **Вариант 2: Создать статическую сборку**

Соберите проект в статические HTML/CSS/JS файлы, которые можно открыть в браузере:

### Шаги:

1. **Соберите проект:**
   ```bash
   npm run build
   ```

2. **Результат будет в папке `/dist`** — это готовые файлы сайта

3. **Способы использования:**
   - **А. Локально:** Откройте `/dist/index.html` в браузере (может не работать из-за CORS)
   - **Б. Простой веб-сервер:** Используйте расширение "Live Server" для VS Code
   - **В. Загрузите на хостинг:** Закиньте папку `/dist` на любой веб-хостинг

**Недостатки:**
- ❌ Нужен веб-сервер для корректной работы
- ❌ Не работает просто при двойном клике на `index.html`

---

## ✅ **Вариант 3: Electron (десктопное приложение)**

Превратите проект в настоящее десктопное приложение для Windows/Mac/Linux.

### Шаги:

1. Установите Electron:
   ```bash
   npm install electron electron-builder --save-dev
   ```

2. Создайте файл `electron.js` с базовой настройкой

3. Соберите приложение:
   ```bash
   npm run build:electron
   ```

4. Получите `.exe` файл для Windows или `.dmg` для Mac

**Плюсы:**
- ✅ Работает как обычная программа
- ✅ Не нужен браузер
- ✅ Можно запустить двойным кликом

**Минусы:**
- ❌ Большой размер файла (100+ МБ)
- ❌ Сложнее в настройке

## 🎯 Моя рекомендация

**Для максимальной простоты используйте Netlify:**

1. Перейдите на [app.netlify.com/drop](https://app.netlify.com/drop)
2. Соберите проект: `npm run build`
3. Перетащите папку `/dist` в окно Netlify
4. Получите ссылку и делитесь ей с кем угодно

**Готово!** 🎉 Теперь тур доступен по одной ссылке без установки программ.

## 🆘 Нужна помощь?

Если возникли сложности с развертыванием:
1. Проверьте, что проект собирается командой `npm run build`
2. Убедитесь, что все фотографии на месте в `/public/tour-images/`
3. Попробуйте другой вариант развертывания

## 📝 Дополнительная информация

### Системные требования для пользователей (при использовании ссылки):
